# 텍스트 강조

**를 앞뒤로 써주면 글자를 굵게 해줌  

**굵게**

*를 앞뒤로 써주면 글자를 기울여줌

*기울임*

~~를 앞뒤로 써주면 취소선을 만들어줌

~~취소선~~

-를 세번 입력하면 수평선을 만들어줌
